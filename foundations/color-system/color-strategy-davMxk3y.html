Color strategy

To build accessible and user-friendly products, it's crucial to grasp the foundation of the Atlas color system. Each level of the Atlas color system builds upon the one before it, ensuring a cohesive and accessible design.

 

What is the Atlas color system

 

The Core color palette and System color scheme work together to create a visually appealing and user-friendly interface. Here's how they relate:

Core color palette

This is our foundation. It's a set of carefully chosen colors that represent our brand and create a cohesive visual identity. Atlas design system encourages using primary, guidance, intelligence, error, and neutral colors to build a balanced and versatile palette. System color scheme is where we apply our palette.

Color scheme

Dictates how those colors are used in your UI in different themes, ensuring proper contrast and hierarchy.

 

 

 

Why is this important?

A color system like the Atlas color system is a valuable tool for designers and engineers. It provides a clear set of guidelines and standardized palettes, ensuring a consistent and harmonious visual language across the product. There are many benefits to this system:

Streamlines the design process
Improves accessibility
Reduces effort needed to build and maintain the UI

Ultimately, it leads to a better user experience that is both visually appealing and functional.
