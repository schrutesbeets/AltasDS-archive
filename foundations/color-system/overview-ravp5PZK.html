Color system

Coordinating consistent, captivating colors

  • Overview
  • Color strategy
  • How to use

 

 

Overview

Color is essential to our brand identity and communication, and is designed with accessibility in mind. We use color intentionally, drawing from traditional color theory while also establishing meanings unique to Tripadvisor, based on how colors are consistently applied across our products and brand presence. This thoughtful approach to color allows us to communicate more effectively and create cohesive, accessible experiences for all users.

 

 

 

Atlas color families

The Atlas color families are derived from the brand colors and optimized for digital product UI. Each family has a set of definitions that provides context on how each color should be used within product UI. This means that each color plays a specific role in product UI.

 

 

 

Atlas system scheme

Core colors are then chosen to create a Atlas color scheme that consists of 57 color roles. The color scheme is created by combining color roles with specific tones from the palette, resulting in a set of semantic color tokens for use across most product UI. Within these roles, colors are intentionally grouped to create harmonious pairings that ensure accessible contrast ratios throughout the product.

 

Dark mode

With Atlas color scheme, both light and dark theme colors are inherited automatically. By using the color roles within the scheme, designs will adapt to both light and dark modes without any extra effort.
