Element Grids

The grid system also allows for grids to be defined and nested within element containers, such as cards, carousel and grid shelf layouts. This allows the pattern to fit within the page constraints, but also gives it flexibility to allow for dynamic UI design. Find more information on their use in the Cards and Shelves pages in this document.

 

 

 

Element Grid Composition

Like Screen Grids, Element Grids are comprised of Columns, Margins, and Gutters.

  • Gutters remain fixed at different screen widths.
  • Columns and Margins can be fluid or fixed at different screen widths. The following is a list of breakpoints and the details that are at each viewport width
