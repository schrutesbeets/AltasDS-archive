Map Pin

A universal, default component with dynamic behaviors that visually represents points of interest (or POIs) on a map, allowing users to interact with and quickly access relevant information.

Also known as Pin, Marker, POI marker

Anatomy & Size

 

Anatomy - Map Pin Maximized Icon

The Map Pin Maximized Icon is used on maps with more than one POI to display the top-ranked POIs.

Anatomy - Map Pin Minimized

The Map Pin Minimized is a simplified representation of lower-priority points of interest (POIs).

Anatomy - Map Pin Anchor

The Map Pin Anchor is a type of pin used to represent the primary listing on a detail page.

 

Size - Map Pin Maximized Icon

Map Pin Maximized Icon has a maximum width of 180px. The Pin title is truncated to one line of text. The Pin subcategory should be truncated to one line if Sponsored is showing, or two lines if Sponsored is not showing

Size - Map Pin Minimized

The Map Pin Minimized measures 16px by 16px

Size - Map Pin Anchor

The Anchor Pin measures 42px by 35px. The icon inside the pin measures 20px by 20px.

 

 

Behavior

Map pins have default and selected content states as well as default and hover states.

Hover states are scaled 120% of the default state

 

Map Pin Minimized Interactive States

Left to right, top to bottom: isolated default, isolated hover, generic default, generic hover, saved isolated default, saved isolated hover, saved default, saved hover

 

Map Pin Maximized Icon Interactive States

Left to right, top to bottom: icon unselected default, icon unselected hover, icon selected default, icon selected hover

 

Map Pin Maximized Currency Interactive States

Left to right, top to bottom: currency unselected default, currency unselected hover, currency selected default, currency selected hover

 

Map Pin Maximized Currency Isolated Interactive States

Left to right, top to bottom: currency isolated unselected default, currency isolated unselected hover, currency isolated selected default, currency isolated selected hover

 

Map Pin Maximized Landmark Interactive States

Left to right, top to bottom: landmark unselected default, landmark unselected hover, landmark selected default, landmark selected hover

 

Map Pin Anchor Interactive States

Left to right: default, hover
