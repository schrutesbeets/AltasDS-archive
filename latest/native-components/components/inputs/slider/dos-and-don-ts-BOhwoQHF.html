Slider

A slider is a control that allows users to make selections from a range between minimum and maximum values.

  • Overview
  • Dos & Don'ts

Properties

 

Histogram

The slider supports an optional histogram to visualize the distribution of results across the range. The slider can also be displayed without the histogram for a simpler, lighter-weight option when distribution data is not needed or available.

 

Range values

The slider always displays a minimum and maximum value to define the available range. These anchors help travelers understand the full scope of options.

 

 

 

Usage

 

Price filter

The price filter uses a dual-handle slider to let travelers set a minimum and maximum price per night. It always displays the full range of available values, with optional histogram data to visualize price distribution and guide selection.

 

Distance filter

The distance filter uses a slider to adjust how far results are shown from a chosen landmark or point of interest.

 

 

 

Behavior

 

Selection

Users can select a value with sliders by dragging the handle. As the value on the slider increases, the section of the track between the minimum value to the knob is filled.

 

 

 

Customizations

The slider’s range in Figma can be customized by modifying the horizontal padding in the auto layout settings of the *select* this layer to adjust layer.

 

To adjust the left and right slider handle position, drill into the component and select the *select* this layer to adjust layer.

 

In the auto layout section of the Design panel, adjust the horizontal space until the slider handle is in your desired position. Do this as need be for the left and right slide handles.

 

 

 

Best practices

 

Provide clear values

Do

Always show current min and max rang values on the track for context.

Don't

Do not hide the min and max values.
