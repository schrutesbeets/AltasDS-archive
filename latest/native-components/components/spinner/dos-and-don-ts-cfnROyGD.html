Spinner

A spinner is a visual loading indicator that a process is in progress but the interface is not yet ready for interaction.

  • Overview
  • Dos & Don'ts

Usage

 

Tasks

Spinners can show the progress of a task, inside a button or loading state, for example.

 

Search

Spinners can communicate the state of search results within the search experience.

 

 

 

Best practices

 

Frequency

Do

Indicate progress on the whole task, not each part.

Don't

Do not show the progress of each section or individual component within a process.

 

Size

Do

Use only predefined spinner sizes for consistency and best practices.

Don't

Do not create custom spinner sizes.

 

Usage

Do

Use a loading state pattern for full page-level loading states with a flow or process.

Don't

Do not use spinners for full page-level loading states as they are meant to serve as lightweight, inline indicators for short delays.
