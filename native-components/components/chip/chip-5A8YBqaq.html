Chip

Chips are compact elements that present contextual, supplemental options to enhance the current journey and encourage action from the user.

Common name alternatives

Pill

 

 

 

Guidelines

Chips are used in groups, appearing together in a set (also known as a chip row), and allow users to filter content, make selections, or trigger actions. They should adapt to the situation and have a clear relationship with the content they represent.

 

 

 

Types

There are three types of chips: Category, Filter, and Action.

 

Category
Filter
Action

 

 

 

Which chip do I use?

 

Category chip

Category chips represent a set of mutually exclusive options. They are used when a user can choose only one option from a predefined set. Selecting a category chip updates the content or state to reflect the chosen category, while automatically deselecting any previously chosen option.

 

Filter chip

Filter chips enable users to refine or narrow down content by applying one or more filters. Unlike category chips, multiple filter chips can be selected at the same time to progressively adjust the results.

 

Action chip

Action chips trigger an action related to the content they represent. They are often lightweight alternatives to buttons, appearing inline or in secondary contexts. They should be direct, clearly communicating the action to the user.
