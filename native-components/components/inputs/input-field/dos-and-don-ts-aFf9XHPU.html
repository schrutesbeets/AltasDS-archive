Input Field

Input fields allow users to enter the number of guests, specify the number of rooms, or select a single date or a range of dates.

  • Overview
  • Dos & Don'ts

Properties

 

Date/Date range

The date field represents a calendar date or range of dates. It includes a fixed leading calendar icon and displays a label or default date range. By default, the field shows “Enter Dates” or a date range starting from today to the next day.

 

For restaurants, the date defaults to “Today”. If a user selects tomorrow the field will show “Tomorrow”, all other dates are displayed as the month’s abbreviation followed by the date

 

Room & Guest

The rooms & guests field represents the number of rooms for hotel bookings and the number of guests for both hotel and experience bookings. It includes a fixed leading icon and displays a default selection. By default, the number of guests is set to two adults, and the number of rooms is set to one.

 

Time

The time field represents a specific time or time range. It includes a fixed leading clock icon and default time selection. Restaurant reservations default to 8:00 PM.

 

Divider

Dividers are required within a segmented input field. They improve readability by clearly distinguishing each segment. They do not accept interaction and are purely decorative.

 

 

 

Usage

 

Segmented input field

Segmented input fields are used as a compact, high-level entry point for users to view and select key pieces of related information, such as dates, rooms/guests, or time. When a user taps on a segment, it opens the corresponding bottom sheet picker.

 

Multi-select input field

Multi-input fields are found within bottom sheet pickers, and provide a detailed interface for making selections. Each single input field within the multi-input allows the user to choose a specific value, and the selections are then reflected back in the segmented input field once applied.

 

 

 

Behavior

 

Responsive layout

When input fields are displayed on larger viewports, the date field width expands while the trailing fields remain fixed.

 

Data formatting

Keep data formatting within input fields to a consistent format. Below details general data format for the United States:

Best practices

 

Icon

Do

Keep the provided icons for each input field type.

Don't

Do not customize the leading icons within the input fields.

 

Data formatting

Do

Reference the correct data formatting style for dates and time.

Don't

Do not stray from the correct data formatting style.

 

Usage

Do

Use multi-input fields within bottom sheet pickers so that users can intuitively switch from one input type to another.

Don't

Do not use segmented input fields within pickers as does not allow users to switch between on input type to another.
