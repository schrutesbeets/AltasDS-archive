Stepper

A stepper is a control that allows users to edit a numeric value with plus and minus buttons to decrease or increase the value.

  • Overview
  • Dos & Don'ts

Properties

 

Label

Labels are required for the stepper. When used they should be short, concise and quickly describe the field’s purpose.

 

Assistive text

A subtitle or assistive text is optional for the stepper. It should be used to clarify the value, min/max quantity, or provide additional context.

 

Minimum value

A minimum value may be set for a stepper. A default value may be set as well. The decrement button is disabled when the value matches the minimum value.

 

Maximum value

A maximum value may be set for a stepper. A default value may be set as well. Once the max value is reached the “add” action is set to a disabled state. The default maximum value is null.

 

 

 

Usage

 

Quantity selection

Use the stepper whenever users need to increase or decrease a number within a defined range. This is commonly used within filters for room and guest selection.

 

 

 

Behavior

 

Text wrapping

Labels and assistive text should be kept short and within one line. However, if longer than one line due to localization, the text will wrap.

 

Tap target

In a stepper, only the plus and minus buttons should be interactive. The label is non-interactive and simply describes the value being adjusted.

 

 

 

Best practices

 

Whole numbers

Do

Always increase and decrease by whole numbers.

Don't

Do not use decimals in the value.

 

Limitations

Do

There should be a limit to the min and max values.

Don't

Do not let the stepper go on without bounds.
