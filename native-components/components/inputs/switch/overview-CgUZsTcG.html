Switch

Switches are controls that allow users to make a binary on/off choice. Their input is immediate and requires no additional action.

Anatomy

 

Anatomy

The switch control is made up of a track and handle/knob. When utilized, it should be paired with a required label and option secondary copy. If using multiple switches in a list format, use a divider to separate them.


The switch remains right aligned to the label text.

 

 

 

States & Appearance

 

States

Switches use state layers for interaction states. The available states are enabled, pressed, focused, selected, and disabled.
