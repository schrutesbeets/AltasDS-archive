Media Container

The media container is a flexible container that displays a variety of visual content.

Ratios

There are three ratios available: 1:1, 3:4, and 16:9. Certain contexts may force one ratio over the other.

 

 

 

 

Media Types

There are fours types of media supported within the media container: Fallback, Image(s), Map, and Lottie animation.

Orientation & Size

 

Orientation

3:4 media containers can be in a portrait or landscape orientation. The orientation depends on the content and its context. Some contexts may need a specific orientation.

 

Border radius

The border radius adjusts according to the size of the image. The radius is 8px for media containers 128x128 and larger. Containers 80x80 and smaller use 8px.

 

Full-width

The media container can be set to the full-width of the screen. In these cases, the container does not have a rounded border radius.

 

 

 

Appearance

 

Dark mode

Images and videos within media containers remains the same across light and dark mode.

 

Fallback content updates color tokens when in light vs dark mode
