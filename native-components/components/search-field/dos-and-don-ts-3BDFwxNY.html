Search Field

Search fields allow users to enter a search query using custom characters and keywords.

  • Overview
  • Dos & Don'ts

Properties

 

Search icon

The leading search icon is static and acts as an indicator as well as an affordance of the field’s purpose.

 

Placeholder text

The placeholder text clarifies what content is filtered when searching.

 

Input text

The input text contains the characters used to filter a list. Ideally, the list is filtered in real time as each character is typed.

 

Trailing action

The trailing action is a clear button by default when the field is focused and receives one or more characters.

 

Cancel & Back button

A cancel button (iOS) or back button (AND) is near the search field in search views or similar contexts where users need a clear option to exit the view.

 

 

 

Usage

 

Global search

The global search field is used when users need to search across the entire platform (destinations, hotels, things to do, etc.).

 

Inline search

The inline search field is used when users need to filter or search within a specific section or module (e.g., reviews, lists, or a page subset).

 

 

 

 

Behavior

 

Overflow

Input text overflows horizontally between the search icon and clear action. While focused, the text shifts to keep the cursor visible. On blur, the text realigns to the start.

 

Scroll

The global search field on Home collapses in size when on scroll.

 

Focused state

Focusing on the search field places the cursor at the start of the placeholder. Once text is entered, the placeholder text is replaced and a trailing clear button is added. Refocusing positions the cursor at the end of the input.

 

 

 

Best practices

 

Search icon

Do

Keep the search icon as the leading icon.

Don't

Do not remove the search icon in any state.

 

Placeholder text

Do

Use clear labeling with placeholder text to indicate what the user should search for.

Don't

Do not use vague or missing placeholder text that leaves users unsure of what they can search for.
