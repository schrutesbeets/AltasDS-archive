Anatomy & Size

 

Anatomy

Search fields contain a required search icon alongside placeholder or inputed text. When enabled, there is a clear button to clear the inputed search as well as a tertiary dismiss button to exit the search experience.

 

Container height

Search fields come in two sizes. Global search fields are 56px while inline search fields are 48px.
