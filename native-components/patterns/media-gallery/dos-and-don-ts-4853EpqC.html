Media Gallery

A media gallery is a repeatable design pattern for displaying collections of images, videos, or other media in a structured and navigable layout.

  • Overview
  • Dos & Don'ts

Properties

 

Navigation header

Media galleries, both standard and mosaic, are displayed with a navigation header displaying the POI name and a trailing action to add a photo.

 

Albums

Media galleries can also be presented with albums. An album has an image thumbnail, an album title, and a number representing how many photos are in the album.

Albums can have the following titles in the following order:

• All
• Room & suite
• Traveler (album = All, type = Traveler)
• Hotel & amenities
• Favorites
• Pool & beach
• Panorama
• Bathroom
• Dining
• Bathroom
• Videos

For detailed display logic, refer to the HOT Media Spec Album Navigation section.

 

 

Display

 

Mosaic layout

The mosaic gallery follows the below layout:


Row 1: Full width media
Row 2: Two vertically stacked media (leading) next to one doubled in height media (trailing).
Row 3: One double height (leading) next to two vertically stacked media (trailing).

Repeat the pattern until no more photos are left. If there are not enough pieces of media to fill a row at the end of the gallery, the last row can be a single full width, or two double height.

 

 

 

Behavior

 

Scrolling

When scrolling through a gallery that contains albums, the album row scrolls until the album titles lock in place below the top nav, creating a tabbed navigation.

 

Interaction

Media in the media gallery can be tapped. Tapping a piece of media will displays more detail about it in the media viewer.

 

 

 

Best practices

 

Layout

Do

Keep the arranged ordering and layout for mosaic galleries.

Don't

Do not change the arranged ordering/layout of the mosaic gallery.
