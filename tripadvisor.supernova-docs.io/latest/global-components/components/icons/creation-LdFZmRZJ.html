Icons

Icons serve as a visual language, providing a consistent and recognizable set of symbols to communicate actions, concepts, or status across an interface.

How to guide

Step 1

Start with a pre-exisiting icon

Duplicate the icon template file. You will find a set of pre-outlined icons to start with in that file. These icons are for you to alter or use aspects of them to help you create your new icon.

 

Step 2

Get inspired. Use Noun Project to jump start inspiration

If you donâ€™t have an icon design in mind, a great way to start is by looking through Noun Project for examples. Signing up for a membership will allow you to download SVG files for free.

 

Step 3

Start designing at 400%

By maintaining this ratio, any changes to the original will be scaled up or down proportionally, which preserves sharp edges and correct alignment when the scale is returned to 100% (48dp).

Tip: Use the pen tool at a 1.5px stroke and keep to the live area within the grid.

 

Step 4

Check, resize, duplicate, and finalize

Refer to the checklist and check off items as you go. Once youâ€™re in a good place with your designs, resize them to 24x24 and see how your designs look at a smaller scale.

  • Does the icon fit within the design guidelines?
  • Does the icon have a 1.5px weight?
  • Does the icon fit within one of the 4 grids?
  • Did you compare with about 4-5 current icons in the set?
  • Was the icon created at 400% scale for pixel perfect accuracy?
  • Did you add alternative names for the icon into the metadata of the artwork?
Reminder: Always keep a version of your icons that are pre-outlined so you can go back and easily edit them.

 

Step 5

Get your icon approved

When youâ€™re ready to take your icons to the next step, get it approved.

Post in the #ta-design-all channel.

The DS and brand team will look over the design, make sure it fits within the icon set, and works within all the design guidelines.

 

Step 6

Outline your design + follow next steps

Submit the icon design as a DS Contribution and include all of the details for the new icon (short description and names)
