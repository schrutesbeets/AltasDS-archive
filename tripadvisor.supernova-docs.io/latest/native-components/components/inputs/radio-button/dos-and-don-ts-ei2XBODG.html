Radio Button

Radio button is a control that allows users to select one option at a time from a set.

  • Overview
  • Dos & Don'ts

Properties

 

Label

A label is required for all radio buttons. They should be as short as possible.

 

Secondary text

Radio buttons can include secondary text (subcopy) beneath the main label to provide additional context. This is helpful when the primary label alone may not fully explain the option.

 

Bubble rating

When filtering by traveler rating, the label is replaced with bubble ratings.

 

Trailing radio button

Radio buttons are trailing by default to prioritize reading the content first, making them ideal for filters, so that users interpret labels before acting.

 

Leading radio button

Place the control before the label so users can quickly scan and select options. Ideal for short, action-focused choices like binary yes/no selections or single-choice lists where the selection itself is the primary focus.

 

 

 

Usage

 

Filters

Use radio buttons in filters when users must select only one option from a list. Radio buttons make the single-choice nature of the selection clear, unlike checkboxes which allow multiple choices.

 

Single select options

Use radio buttons for single-select options where users must choose one mutually exclusive choice from a set. Selecting a new option automatically deselects the previous one, making the single-choice behavior clear.

 

 

 

Behavior

 

Label overflow

Labels should be kept short, but will wrap if they are wider than the container. The radio button remains aligned to the top of the text box.

 

Text wrapping

By default, content starts to truncate after a certain number of lines as follows:

 

Label

Secondary text

2 lines

5 lines

 

Single selection

Selecting a radio button will automatically deselect the previously selected item as only one radio button can be selected at a time.

 

Tap target

The entire radio button container is the tap target when selecting the radio button. Tapping anywhere on the item should activate the control.

 

 

 

Best practices

 

Selection

Do

Radio buttons should only be used when a single element can be selected.

Don't

Do not use radio buttons when multiple options can be applied simultaneously.

 

Alignment

Do

Keep radio button alignment consistent when listing out related items.

Don't

Do not mix leading and trailing radio buttons in one list.
