Stepper

A stepper is a control that allows users to edit a numeric value with plus and minus buttons to decrease or increase the value.

  • Dos & Don'ts

Anatomy

 

Anatomy

The stepper component has two secondary icon buttons on each side to increase and decreases the value in the middle. When utilized, it should be paired with a required label and optional limit subcopy.


The stepper remains center aligned to the text content.

States

 

Interactive states

Steppers use state layers for interaction states. The available states are enabled, pressed, focused, and disabled.
