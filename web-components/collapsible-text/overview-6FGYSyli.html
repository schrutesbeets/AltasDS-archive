Collapsible Text

The combination of a text block and button, used to truncate content when collapsed and reveal content when expanded

  • Overview
  • Dos & Don'ts

 

Variants

There are four main variants of Collapsible Text: Minimal, Standard, Medium, and Extended.

Type minimal
Type standard
Type medium
Type extended

 

 

 

Anatomy & Size

Collapsible Text consists of a limited number of smaller components, and generally has variable dimensions.

 

Anatomy

Collapsible Text consists of a text block and an accompanying Borderless Button.

 

Height

The height of Collapsible Text is varies among variants, is based on the both the size of the variant text as well as the number of lines of text the variant contains. The height of the expanded state is similarly variable.

 

Width

The width of Collapsible Text is variable based on the number of page grid columns the component spans.

 

 

Behavior

Collapsible Text components have the ability to expand to reveal additional content, or collapse to hide additional content.
