Web Components

Atlas provides an extensive set of components for use in building larger web experiences and patterns.

Supported Components

Supported components are in greater parity between Figma and web code. They are currently being optimized by the DS team, and are intended to receive ongoing bug and contribution support.

Avatar

A graphical representation of a user: usually a photo, illustration, or initial.

Buttons

Buttons are controls that let users take action, make choices, and move forward. Also known asÂ action, call to action, or CTA.

Checkbox

An input for choosing from predefined options. When used alone, it gives a binary choice (checked/unchecked).

Checkbox with Label

A checkbox accompanied by label text.

Collapsible List

A list of topic titles that reveal or hide associated sections of content.

Also know as Accordion, Arrow toggle, Expander

Collapsible Text

The combination of a text block and button, used to truncate content when collapsed and reveal content when expanded

Facepile

The representation of multiple traveler identities.

Horizontal Scroll Bar Button

The combination of Circular Button and a gradient used to scroll elements within a row, bar, or carousel.

Image

An element for embedding images of fixed proportions.

Inline Alert

A way of informing the user of important changes in a prominent way.

Also known as Notification, Feedback, Message, Banner, Callout

Modal

A modal is an interface element that appears over other content. It requires an interaction from the user before they can return to whatever is underneath.

Also known as Dialog, Popup, Modal Window

Progress Bar

A representation of a userâ€™s progress through a series of discrete steps. Also known as Progress Tracker or Progress Meter

Selector

A toggle button used for single and multi-selection inputs, specifically within question and answer flows.

Sheet

A panel which slides out from the edge of the screen.

Also known as drawer or tray

Slider

A form control for choosing a value within a preset range of values.

Also know as a Range Input.

Spinner

A visual indicator that a process is happening in the background but the interface is not yet ready for interaction.

Also known as Loader, Loading

Switch

A control used to switch between two states: often on or off.

Also known as Switch, Lightswitch, Toggle button

Switch Row

A text label and/or description, accompanied by a Switch.

Stepper with Label

A control for editing a numeric value with buttons for decrementing / incrementing.

Tab Bar

Tab Bars are a way of navigating between multiple panels, reducing clutter and fitting more into a smaller space.

Also known as Tabbed Interface

Text Links

A link is a reference to a resource. This can be external (e.g. a different web page) or internal (e.g. a specific element in the current document).

Also known as Anchor, Hyperlink

Toast

A type of alert which appears in a layer above other content, visually similar to a mobile or desktop push notification.

Also known as: Snackbar, Alert

Tooltip

Tooltips are small overlays that appear when users hover over, focus on, or tap an element. They provide context or explain the function of a piece of UI.

Also known as Screentip, Infotip, Hint
